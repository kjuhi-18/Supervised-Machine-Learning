# 🧠 What is Linear Regression?

Linear Regression is used to model the relationship between:

- **Independent Variable (X)**  
- **Dependent Variable (Y)**

It finds the **best-fit line**:

\[
Y = mX + c
\]

Where:

- **Y** = Predicted output  
- **X** = Input feature  
- **m** = Slope  
- **c** = Intercept
