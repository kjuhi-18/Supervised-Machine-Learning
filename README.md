# 📌 What is Supervised Machine Learning?

Supervised Machine Learning is a type of learning where:

- The model is trained on **labeled data**
- It learns the relationship between **inputs (X)** and **outputs (Y)**
- Then it predicts outputs for new unseen data

👉 Example:
- Input: Study hours  
- Output: Exam score  

---

# 🎯 Types of Supervised Learning

## 1️⃣ Regression

Used when the output is **continuous (numeric)**

✔ Predict salary  
✔ Predict price  
✔ Predict temperature  

---

## 2️⃣ Classification

Used when the output is **categorical (labels)**

✔ Spam or Not Spam  
✔ Disease or No Disease  
✔ Buy or Not Buy

# 📂 Algorithms Covered

## 📈 Regression Algorithms

### 1️⃣ Linear Regression
- Simple relationship between variables  
- Straight line model  

---

### 2️⃣ Multiple Linear Regression
- Uses multiple features  
- More realistic predictions  

---

### 3️⃣ Polynomial Regression
- Handles non-linear data  
- Curve fitting  

---

### 4️⃣ Ridge & Lasso Regression
- Regularization techniques  
- Prevent overfitting  

---

## 🤝 Instance-Based Learning

### 5️⃣ K-Nearest Neighbors (KNN)
- Distance-based learning  
- Used for classification & regression  

---

## 🌳 Tree-Based Models

### 6️⃣ Decision Tree
- Rule-based model  
- Uses splitting (ID3, CART)  

---

### 7️⃣ Logistic Regression
- Probability-based classification  
- Uses sigmoid function  

---

## 📊 Classification Algorithms

### 8️⃣ Naive Bayes
- Based on probability & Bayes theorem  
- Includes:
  - Gaussian NB  
  - Multinomial NB  
  - Bernoulli NB  

---

### 9️⃣ Support Vector Machine (SVM)
- Finds best separating boundary  
- Uses margin & kernel trick  

---

### 🔟 Random Forest
- Ensemble of multiple trees  
- Reduces overfitting
